Sorting

1. Stability in sorting - if two values are same wrt to sorting variable, it should be in input order
   stable - Bubble sort,  Insertion sort, Merge sort
   unstable - Selection sort, Quick sort, heap sort
2. Bubble sort - Theta (n^2), 
3. Selection sort - Theta (n^2), basic idea of heap sort
4. Insertion sort - Worst case - Big(n^2), Best case - Big(n^2), Tim sort (python) - Insertion + Merge
